# User Guide — LavaSMSID

Panduan ringkas ini membantu pengguna sekolah memakai LavaSMSID sesuai role masing-masing.

## Akses Aplikasi

1. Buka URL aplikasi sekolah.
2. Login menggunakan akun resmi dari administrator.
3. Gunakan menu yang tersedia sesuai role.
4. Logout setelah selesai, terutama pada perangkat bersama.

## Dashboard

Dashboard menampilkan ringkasan informasi sesuai role, seperti jadwal, notifikasi, tagihan, absensi, nilai, PPDB, atau laporan. Jika menu tidak muncul, hubungi admin sekolah untuk memeriksa role dan permission.

## Penggunaan Umum

- Gunakan pencarian, filter, dan pagination untuk menemukan data.
- Periksa kembali data sebelum menyimpan perubahan.
- Unggah file hanya jika format dan ukuran sesuai ketentuan aplikasi.
- Jangan membagikan data pribadi siswa, guru, staff, orang tua, atau alumni kepada pihak tidak berwenang.

## Per Role

- **Kepala Sekolah**: memantau dashboard, laporan, dan ringkasan kinerja sekolah.
- **Waka Kurikulum**: mengelola data akademik, jadwal, kelas, nilai, dan rapor sesuai akses.
- **Waka Kesiswaan**: memantau siswa, absensi, kegiatan, dan data pembinaan.
- **Guru / Wali Kelas**: mengelola jadwal, absensi, nilai, dan informasi kelas yang menjadi tanggung jawabnya.
- **Siswa**: melihat informasi akademik, jadwal, nilai, absensi, tagihan, dan pengumuman.
- **Orang Tua / Wali**: memantau informasi siswa yang terhubung, termasuk absensi, nilai, tagihan, dan pengumuman.
- **Bendahara**: mengelola tagihan, pembayaran, dan laporan keuangan sesuai permission.
- **Panitia PPDB**: memeriksa pendaftar, berkas, status seleksi, dan proses konversi sesuai kebijakan sekolah.
- **Pembimbing PKL**: memantau penempatan, logbook, monitoring, dan nilai PKL.
- **Admin BKK**: mengelola alumni, tracer study, lowongan, dan mitra perusahaan.

## Bantuan

Jika menemukan data salah, akses hilang, atau error aplikasi, catat waktu kejadian, menu yang digunakan, dan tangkapan layar bila aman. Kirim laporan ke admin sekolah tanpa menyertakan password atau secret key.
